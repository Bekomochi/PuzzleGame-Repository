# このゲームの仕様について  
同じ画像のピースを三つ以上繋げて消すカジュアルパズルゲームです。  
ピースを繋げることに距離の制限はないので、行動制限のないマッチ3パズルという仕組みです。  

# 操作方法  
指一本、またはタッチペンを使用します。  
画面をタップ、もしくはフリックして操作します。  

# ダウンロードするもの  
特にありません。
